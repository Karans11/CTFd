What is Wiz Cloud Hunters?

Wiz Cloud Hunters is a Capture The Flag competition designed for developers, security engineers, and cloud practitioners. Challenges are based on real-world cloud security scenarios covering AWS, Azure, GCP, Kubernetes, and container security.

Protect Everything You Build and Run in the Cloud

Wiz is the Cloud Security Platform that enables security, dev, and devops teams to work together in a self-service model built for the scale and speed of cloud development.

Challenge Categories
